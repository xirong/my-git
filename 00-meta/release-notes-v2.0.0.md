# v2.0.0: Modern Git & AI-Native Engineering Handbook

## Highlights

- 更新仓库定位，面向现代工程团队与 AI 编程时代的 Git / GitHub 实战场景
- 新增 AI Native Git Workflow、AI 代码 Review、AI 变更审查实战样例
- 新增团队 Git 工作流指南，覆盖 GitHub Flow、GitLab Flow、Gitflow、Trunk-Based Development、Release Branch 等选型
- 新增 GitHub 工程治理内容，覆盖分支保护、Rulesets、CODEOWNERS、Merge Queue、Actions、Release、安全扫描
- 新增 Git 高频事故处理手册，覆盖误提交、reset、rebase、force push、secret 入库等常见问题
- 新增大仓库实践内容，覆盖 partial clone、sparse checkout、Git LFS、worktree、repo maintenance
- 新增可复制模板，包括 PR、Issue、Code Review、AI Review、commit message、release note、hotfix、branch naming
- 新增大厂工程实践案例目录，用公开资料整理 Google、Microsoft、Meta、Uber、Netflix、Shopify、Slack 等工程经验

## Why v2.0

v2.0 聚焦三个变化：

1. Git 的使用场景从个人命令学习，扩展到团队协作、Review、发布、恢复和工程治理
2. AI 编程让代码生成更快，也让 diff 审查、commit 拆分、验证和回滚更重要
3. 中文工程团队需要更多能直接落地的 Git / GitHub 协作模板和判断路径
