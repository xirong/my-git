# Contributing

感谢你愿意一起维护这个仓库。

这个仓库的新方向是现代工程团队 Git / GitHub 实战手册，欢迎贡献真实问题、团队实践、事故恢复经验、AI 编程工作流和可复用模板。

## 推荐贡献方式

1. 先开 Issue 说明你想补充的问题
2. 说明目标读者是谁
3. 说明这个内容解决什么真实问题
4. 提交 PR 时保持改动聚焦

## 内容要求

- 优先贡献实战判断，不只补链接
- 命令必须能被读者理解风险
- 涉及危险操作要给出前置检查
- 涉及 GitHub 功能要尽量引用官方文档
- 模板类内容要可直接复制使用

## PR 要求

- 每个 PR 尽量只解决一个主题
- 新文章请放入对应目录
- 新内容请遵循 [Content Style Guide](content-style-guide.md)
- 修改旧文章时，请尽量保留历史链接可用
